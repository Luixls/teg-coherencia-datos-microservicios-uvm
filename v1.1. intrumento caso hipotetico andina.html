Versión demostrativa prellenada con el caso hipotético Comercial Andina Digital, C.A. de la sección 4.9 del TEG. Este archivo permite registrar el proceso distribuido, diagnosticar su madurez transaccional, evaluar la aplicación de los lineamientos, seleccionar un patrón transaccional, revisar indicadores y generar un reporte imprimible para la toma de decisiones.

1. Datos generales de aplicación

Identificación del proceso evaluado
Complete esta sección antes de aplicar los lineamientos. El instrumento puede aplicarse a un proceso real, un flujo piloto o un caso hipotético de validación.

2. Diagnóstico inicial

Madurez transaccional y criticidad
Seleccione el nivel MMTM que más se aproxime al estado actual del proceso u organización. Luego evalúe la criticidad del flujo.

Criterios de criticidad del proceso

Resultado de criticidad: Baja

3. Evaluación de los diez lineamientos

Escala de aplicación y evidencias
Use la escala: 0 = No iniciado, 1 = Básico, 2 = Parcial, 3 = Aplicado, 4 = Evidenciado. La columna de evidencia permite dejar registro del documento, tablero, log, matriz o responsable que sustenta la evaluación.

Porcentaje de aplicación: 0% · Interpretación: Inicial

Lineamiento Dimensión Estado Evidencia / observación Acción pendiente

4. Selección orientativa de patrón transaccional

Decisión según contexto del proceso
Esta sección no reemplaza una revisión arquitectónica. Su propósito es orientar la decisión inicial con base en las condiciones del proceso evaluado.

Patrón sugerido: —

Complete los criterios para generar la recomendación.

5. Indicadores operacionales de evaluación

KPIs referenciales de coherencia de datos
Ingrese los valores actuales del proceso evaluado. El instrumento compara cada valor con una meta referencial y calcula cuántos indicadores están dentro del rango esperado.

6. Reporte final de aplicación

Resumen imprimible
Pulse “Generar reporte” después de completar las secciones. Luego puede usar “Imprimir reporte” para guardarlo como PDF desde el navegador.

El reporte se generará con base en la información registrada en el instrumento.
